Detailed Description

Test Suite for FW State Machine Module and FW Procedure Module.

This file defines the main program for the test suite. The test suite executes a number of test cases in sequence and reports the outcome of each test case. Test cases for the state machine module are defined in FwSmTestCases.h. Test cases for the procedure module are defined in FwPrTestCases.h.
